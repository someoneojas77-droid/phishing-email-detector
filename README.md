# Phishing Email Detector

## Overview
This project is a phishing email detection system developed as part of the Bring Your Own Project (BYOP) for the course *Fundamentals of Artificial Intelligence and Machine Learning* at VIT Bhopal University.

The system analyzes email text and classifies it as either phishing or safe. It is designed to help users identify suspicious emails and improve cyber awareness.

---

## Problem Statement
Phishing emails are a major cybersecurity threat. They are designed to trick users into sharing sensitive information such as passwords or personal data. 

Students are especially at risk because they often receive emails related to internships, job offers, and academic opportunities. These emails can appear genuine, making them difficult to identify.

This project aims to provide a simple solution to detect such emails using basic AI and machine learning techniques.

---

## Features
- Classifies emails as phishing or safe  
- Uses machine learning (Naive Bayes)  
- Simple command-line interface  
- Easy to understand and use
